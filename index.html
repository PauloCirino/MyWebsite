Bias vs Variance

The bias and variance tradeoff is a central problem in supervised learning. A model with high bias is overfitted to trainning data, which is harmfoul because it does not generalizes well to unseen data. Contrarily, a model with high variance is one that is underfitted to the trainning data, because it does not approximate it well enough. This app was created to explore and exemplify this conundrum, allowing its user to play around different models and datasets.

Multi-GPU Kmeans

This is a project to showcase the usage of Tensorflow in a multi-GPU enviroment. The computation of these algorithms was implemented in a way that distribuited different data points to different GPUs, this was achived by breaking down the equations that are used to update the new center and weighing it by each GPU.
